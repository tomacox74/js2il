# JavaScriptRuntime (JS2IL)

This package is the runtime support library used by JS2IL-compiled assemblies.

## Hosting compiled JavaScript from C#

If you want to compile JavaScript into a .NET assembly and then **host that compiled module as a library** (typed or dynamic), see:

- https://github.com/tomacox74/js2il/blob/master/docs/hosting/Index.md
